# wrchive
### Revive Weatheradio Canada on PC (Windows, with planned MacOS/Linux)

# First time Setup process (WINDOWS)
### 1. Download the .zip
### 2. Extract and run wrchive.exe -- If you get a defender warning, do not worry! This is because I'm a broke teenager who can't afford a few hundred bucks a year for a license. wrchive is safe.

# Beginning and running a cycle
### Each time you enter wrchive, you will need to input the rss feed link for the location you want. Once this has been started, a cycle contained 7 day forecasts and TONED SAME alerts will begin containing all feeds inputted

# Getting feeds
### 1. To get rss feeds, visit https://weather.gc.ca/canada_e.html
### 2. Select a location from here or search at the top for a location.
### 3. Once you have found a location, scroll to the bottom. You should see two buttons, an orange RSS feed button and a general storm looking button.
### 4. Select the orange button and then select weather (not alerts!)
### 5. Ensure the feed's URL looks something like this -> (https://weather.gc.ca/rss/weather/53.536_-113.494_e.xml), and ensure the feed is in ENGLISH (if you are not sure, english feeds end in _e.xml and french ends in _f.xml)
### 6. Copy the URL
### 7. In wrchive, click on "Add Locations" and then paste in the URL. Make sure that you only paste one URL at a time, and that there are no extra characters added, then press enter.
### 8. If done correctly, you should see text confirming a valid feed has been added. A cycle should also begin if it is the first inserted feed.

# Removing feeds
### 1. In wrchive, click on "Remove Locations"
### 2. Follow the prompt below on screen/
### 3. Paste the feed URL of the feed you would like to remove, then hit enter.
### 4. This feed will now be removed
NOTE: The feed that is currently playing cannot be removed until it is no longer being played during the cycle.

# Enabling/Disabling Alerts
### 1. In wrchive, click on "Enable/Disable Alerts"
### 2. Follow the prompt below on screen.
### 3. Type the three letter EAS CODE of the alert you would like to disable -> Ex: TOR for Tornado Warning
### 4. Now, if an alert of that code is in cycle, it will be skipped.
NOTE: This menu also allows you re-enable all alerts and to disable all alerts

# Extras
### 1. In Settings, you can disable French. This disables the french in both alerts and in the forecasts.
### 2. You can also toggle on 2x volume from the same place.
